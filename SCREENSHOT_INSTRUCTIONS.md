# Screenshot Instructions for README

## How to Take the Perfect Screenshot

1. **Open the App**
   - Navigate to: http://localhost:3001
   - Make sure the app has fully loaded with the new Grid_icon.png logo

2. **Setup the View**
   - Click on "Project Alpha" base in the sidebar (should be expanded)
   - Select the "Tasks" table
   - Make sure you're on the "Grid" view (button should be active/glowing)
   - The data grid should show some sample tasks

3. **Take the Screenshot**

   **Option A: Windows Snipping Tool (Recommended)**
   - Press `Windows + Shift + S`
   - Select "Rectangular Snip"
   - Capture the entire browser window including:
     - The sidebar with logo
     - The navbar at top
     - The view switcher (Grid/Kanban/Calendar/Gantt/Form)
     - The data grid with tasks
   - The screenshot will be copied to clipboard
   - Open Paint or any image editor and paste (Ctrl+V)
   - Save as: `C:\Users\meyea\Documents\GitHub\Gridblock\public\screenshot.png`

   **Option B: Browser Screenshot**
   - Press `F12` to open DevTools
   - Press `Ctrl + Shift + P`
   - Type "screenshot" and select "Capture screenshot"
   - Save as: `C:\Users\meyea\Documents\GitHub\Gridblock\public\screenshot.png`

   **Option C: Full Page Screenshot (Best for Showcase)**
   - Press `F11` for fullscreen
   - Press `Windows + Shift + S` and capture
   - Press `F11` again to exit fullscreen
   - Save as: `C:\Users\meyea\Documents\GitHub\Gridblock\public\screenshot.png`

4. **Verify Screenshot**
   - Make sure the screenshot shows:
     ✅ Your new Grid_icon.png logo in the sidebar
     ✅ The elegant dark background (black with burgundy/navy)
     ✅ Rose gold (#EDC5BB) accent colors
     ✅ Glass effect panels
     ✅ The hierarchical sidebar (Spaces > Bases > Tables)
     ✅ The view switcher at the top
     ✅ Some sample data in the grid

5. **Update README**
   - Once saved, the README will automatically show the screenshot
   - The file is already referenced in README.md as `./screenshot.png`

## Recommended Screenshot Specs

- **Format**: PNG
- **Dimensions**: 1920x1080 or 1280x720 (16:9 ratio)
- **Quality**: High (not compressed)
- **Location**: `public/screenshot.png`

## Tips for Best Results

- Use a clean browser window (no bookmarks bar)
- Zoom level at 100% (Ctrl+0)
- Make sure the sidebar is expanded
- Have at least 3-4 tasks visible in the grid
- Capture during the day when your screen brightness is optimal
- The glassmorphic effects look best with content behind them
